Demonstration of Traceroute tool

Traceroute is a computer network diagnostic tool originally written by Van Jacobson to determine whether routing problems exist on the network. It can be used to determine the path IP packets are taking from your computer to the destination computer. Windows uses an ICMP only traceroute utility called 'tracert'. Unix based systems use a UDP based traceroute by default but it can also use ICMP by using -I option.
